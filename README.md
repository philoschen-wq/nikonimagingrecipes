# Nikon Imaging Recipes

> Nikon Z 系列靈活色彩 (Flexible Picture Control) 成像配方展示站 — by Philos

## Tech Stack

- **Vite + React 18**
- **React Router**
- **Tailwind CSS**
- 純 JavaScript（未使用 TypeScript）

## 開始使用

需要 Node.js 18+。

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 打包
npm run build

# 本地預覽打包結果
npm run preview
```
